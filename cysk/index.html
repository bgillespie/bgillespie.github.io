Crates You Should Know

Some of these from this Reddit thread.

Libraries

Niceties

  • anyhow - Easy idiomatic error handling.
  • thiserror - A convenient derive macro for the standard library's std::error::Error trait.
  • lazy_static

Data structures

  • itertools - Extra iterator adaptors, iterator methods, free functions, and macros.
  • boolinator - Provides the Boolinator trait, which lets you use Option and Result-style combinators with bools.
  • bytes - A utility library for working with bytes.
  • chrono - Date and time library.
  • dashmap - Concurrent map.
  • regex - Parsing, compiling, and executing regular expressions.
  • slice-deque - A double-ended queue that Derefs into a slice, also known as a ring buffer or circular buffer.

System tools

  • nix - Rust bindings to *nix APIs.
  • winapi - Rust bindings to Windows APIs.
  • socket2 - Utilities for handling sockets.

Input / Output

  • serde - A framework for serializing and deserializing data structures efficiently and generically.
    • serde-json
  • structopt - Parse command line arguments by defining a struct.
  • clap

Logging, debugging, observability

  • log
  • fern - Simple, efficient logging.
  • tracing - A framework for instrumenting Rust programs to collect structured, event-based diagnostic information.

Async

  • crossbeam - A set of tools for concurrent programming.
  • simple-mutex - More efficient than std::sync::Mutex and simpler than parking_lot::Mutex.
  • parking_lot - Smaller, faster, more flexible synchronization primitives than those in std.
  • rayon - Data parallelism library.
  • flume - An MPMC multi producer, multi consumer channel.
  • futures - Providing the foundations for asynchronous programming.
  • tokio - A runtime for writing reliable, asynchronous, and slim applications.
  • async-std - An async version of std. It provides all the interfaces you are used to, but in an async version and ready for Rust's async/await syntax.
  • ryu - Quickly convert floating point numbers to decimal strings.

CLI tools

  • exa - Replacement for ls.
  • bat - cat replacement.
  • fd - find replacement.
  • ripgrep - grep replacement.
  • sd - sed replacement.
  • dust - du replacement.
  • ytop - top replacement.
  • procs - ps replacement.
  • bandwhich - iotop replacement.
  • hyperfine - time replacement.
  • tokei - Counts lines of code in a codebase, SLOC, plaintext, language code etc.
  • flamegraph - Code profiler. Flamegraph generator with additional support for Cargo projects. It can be used to profile anything, not just Rust projects!

Cargo plugins

  • cargo-geiger - detects unsafe usages in crate and its dependencies.
  • cargo-edit - manage create dependencies in Cargo.toml.
  • cargo-upgrade - upgrade cargo-installed executables.

Other

  • rustfmt - Standard formatting for Rust code.
  • clippy - Linter for Rust code.
  • Evcxr
